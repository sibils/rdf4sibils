SIBiLS RDF files download

The RDF data is available as a collection of files. Some files are compressed and / or tar archives for practical reasons. It is provided in Turtle format (ttl extension) and includes the ontology, void metadata, query examples and data.
The content reflects what is made available at our sparql endpoint.
  • data_medline_ttl_gz.tar - the archive contains multiple compressed ttl.gz files describing a subset of MedLine annotated publications
  • data_pmc_ttl_gz.tar - the archive contains multiple compressed ttl.gz files describing a subset of PMC annotated publications
  • terminologies_ttl.tar - the archive contains multiple compressed ttl files describing the concepts used for annotating the publications
  • citing-sources.ttl - the file contains the list of sources (SIB resources) that cite the publications of SIBiLS RDF dataset
  • ontology.ttl - the file contains a formal desciption of the terms used to descibe publications, annotations and terminologies
  • queries.ttl - the file contains a set of SPARQL example queries
  • void-sibils.ttl - the file contains some metadata about the content of the RDF
